This project is part of a regression analysis Kaggle competition where the goal is to predict a continuous target variable using machine learning models. The dataset was preprocessed by removing irrelevant columns, handling missing values, encoding categorical variables using one-hot encoding, and performing feature engineering on the timestamp data. Two regression approaches were implemented: XGBoost Regressor and Random Forest Regressor. Both models were tuned using RandomizedSearchCV with cross-validation and evaluated using RMSE on a holdout validation set. Final predictions were generated for the Kaggle test set and submitted for evaluation. The best performing model was the tuned XGBoost model, which achieved the lowest RMSE on both public and private leaderboards.
